About Cryptogram

Cryptogram is software for password-protecting images to provide greater control over access to the images. The software takes regular JPEG images as input and produces images that look like random noise. With the correct password, the original images can be restored. The encoding format is designed to be robust to some level of JPEG compression. This means you can upload Cryptogram images to various social media services and the underlying encoded data should not be lost.
